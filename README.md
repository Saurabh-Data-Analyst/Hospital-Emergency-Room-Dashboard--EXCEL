# 🏥 Hospital-Emergency-Room-Dashboard--EXCEL
Hospital ER dashboard in Excel analyzing patient data, wait time, and performance metrics for better decision-making.

## 📌 Project Overview -

This project analyzes Hospital Emergency Room data to uncover meaningful insights related to patient flow, waiting time, and overall service efficiency. The dashboard helps in understanding key operational metrics and improving decision-making.

## 🎯 Objective -

The objective of this project is to identify factors affecting patient satisfaction and hospital performance, and to provide data-driven insights for better management.

## 🛠 Tools & Technologies Used -

* Microsoft Excel
* Data Cleaning
* Data Visualization

## 📊 Key KPIs -

* Total Number of Patients
* Average Waiting Time
* Patient Satisfaction Score
* Admission vs Non-Admission
* Gender-wise Patient Distribution

## 📈 Key Insights -

* Higher waiting time leads to lower patient satisfaction
* Majority of patients were attended within the expected time
* Patient distribution is almost equal across genders
* Some departments handle a higher volume of patients

## 🔍 Process -

1. Data Cleaning (handling missing values, formatting)
2. Data Analysis (identifying trends and patterns)
3. Dashboard Creation using Excel

## 📁 Dataset -

* Source: This dataset used in this project is sourced from a Youtube tutorial for learning purpose.

## 💼 Business Impact -

* Helps reduce patient waiting time
* Improves patient satisfaction
* Supports better hospital resource planning
